Delete Notes
------------

To delete a Note, use the following query:

.. code::

    DELETE /v3/notes/{noteID}

For example, the following query will delete the Note with ID 1:

.. code::

    DELETE /v3/notes/1

JSON Response:

.. code:: json

    {
      "message": "Deleted",
      "status": "Success"
    }
