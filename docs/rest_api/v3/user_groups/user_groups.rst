===========
User Groups
===========

A user group is a list of users that can be assigned to `Workflow Cases <https://docs.threatconnect.com/en/latest/rest_api/v3/case_management/cases/cases.html>`_ and `Workflow Tasks <https://docs.threatconnect.com/en/latest/rest_api/v3/case_management/tasks/tasks.html>`_.

Endpoint: ``/api/v3/security/userGroups``
