=====
Users
=====

Users can perform a variety of actions in ThreatConnect depending on their user account type and their System and Organization role. In addition, users can be assigned to `Workflow Cases <https://docs.threatconnect.com/en/latest/rest_api/v3/case_management/cases/cases.html>`_` and `Workflow Tasks <https://docs.threatconnect.com/en/latest/rest_api/v3/case_management/tasks/tasks.html>`_.

Endpoint: ``/api/v3/security/users``
