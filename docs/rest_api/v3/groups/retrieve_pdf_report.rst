Retrieve a PDF Report for a Group
---------------------------------

Send a request in the following format to retrieve a PDF report for a Group:

.. code::

    GET /v3/groups/{groupId}/pdf

You can retrieve PDF reports for the following Group types:

- Adversary
- Attack Pattern
- Campaign
- Course of Action
- Event
- Incident
- Intrusion Set
- Malware
- Report
- Tactic
- Threat
- Tool
- Vulnerability
