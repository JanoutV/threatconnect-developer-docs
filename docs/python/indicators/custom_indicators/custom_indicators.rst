Custom Indicators
-----------------

Custom Indicators types can be created in ThreatConnect and allow you to capture specific data points that will be helpful as you build intelligence. To view a list of the custom Indicators available on your instance of ThreatConnect, refer to the section on `Retrieving Custom Indicator Types <https://docs.threatconnect.com/en/latest/python/indicators/indicators.html#retrieving-custom-indicator-types>`_ below or the API call described `here <https://docs.threatconnect.com/en/latest/rest_api/indicators/indicators.html#retrieve-available-indicator-types>`_.
