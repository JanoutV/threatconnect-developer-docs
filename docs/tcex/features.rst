Features
--------

+ Command-line interface (CLI) tools to assist in creating, updating, testing, and packaging an App
+ Automatic authorization for hash-based message-authentication code (HMAC) and token-based authentication
+ App logger that supports both filesystem and API logging (:ref:`logging`)
+ Common methods for proper App execution: exit codes (:ref:`exit`), exit messages (:ref:`message_tc`), and more
